One simulation, four teaching views. Use Student Quick Mode for a clean classroom run, EBB Classroom Mode for ABCD/SAGE/PRF integration, Technical/UCP Mode for coherence and constructor-theory interpretation, and Instructor Mode for teaching notes.

EBB Classroom Use

This tool helps students practice the movement from OS1 moral salience to OS2 ethical repair. It connects PRF, SAGE, ABCD ethical attention, moral perception, institutional pressure, and repair responsibility.

Core classroom question: What does the system see first, what does it miss, and what repair practice would allow vulnerability to become morally visible before action?

Scenario

A patient is scheduled for discharge but appears confused about medication instructions and lacks stable home support. A nurse raises concern. The physician, administrator, repair node, and outside advocate interpret the signal through different PRF lenses and institutional pressures.

Question: Does the patient’s vulnerability become visible as a moral claim, or is it converted into workflow friction?

Instructor Notes and Canvas Handoff

Recommended sequence

Run Baseline, Efficiency-Dominant, Right Words Wrong Seeing, Fearful Hierarchy, and Repair Intervention. After each run, ask students to identify the morally relevant signal, the distortion mechanism, and the repair move.

Canvas-ready reflection prompt

In 150–250 words, explain how this simulation illustrates the claim that moral action begins before action. Identify one variable or PRF lens that changed what became morally visible, and propose one repair practice that would reduce perceptual harm.

EBB ecosystem links

PRF Agent Library supports the PRF lens layer. Pair this simulation with MCCTS to help students translate the simulation run into an OS1→OS2 argument, reflection, and repair plan. It can also be paired with Capability Lab and the Perceptual Harm paper as a classroom bridge from moral perception to ethical repair.

MS-3 is a teaching simulation, not a clinical, psychological, or institutional diagnostic tool. It models salience-forming conditions for classroom discussion.
